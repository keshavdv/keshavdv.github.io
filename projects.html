Here are some past random projects I've worked on...

DroidAuto

DroidAuto is an attempt to reverse-engineer the Android Auto protocol and create a working emulator. The end goal is to allow an Android tablet to act as a display for Android Auto and eventually be mounted inside a car. Still very much a work in progress.

Spotifly

GitHub

Spotifly allows you to turn any device into a Spotify Connect enabled endpoint. Simply start the spotifly server and the device will show up on your phone as a Connect device.
